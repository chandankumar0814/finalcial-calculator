# Financial Calculators - Premium Investment Tools

A professional-grade, mobile-optimized suite of financial calculators designed for investors, traders, and retirement planners. Built as a Progressive Web App (PWA) for a native app-like experience on both mobile and desktop.

![App Screenshot](logo.png)

## 🚀 Key Features

The app includes 8 powerful calculators to help you make informed financial decisions:

1.  **CAGR Calculator**: Calculate Compound Annual Growth Rate for any investment.
2.  **SIP Calculator**: Project future returns for Systematic Investment Plans with interactive charts.
3.  **Lumpsum Calculator**: Estimate growth for one-time investments.
4.  **Goal-based SIP**: Calculate the monthly investment required to reach your financial targets.
5.  **Goal-based Lumpsum**: Find out the initial capital needed today to achieve a future goal.
6.  **Retirement Planner**: Comprehensive corpus calculation based on expenses, inflation, and life expectancy.
7.  **SWP Calculator**: Analyze Systematic Withdrawal Plans and portfolio sustainability.
8.  **Position Sizing**: Essential risk management tool for traders to calculate share quantity and risk-reward ratios.

## 📱 Progressive Web App (PWA)

This application is fully PWA-compliant, allowing you to:
- **Install as an App**: Add it to your Home Screen on iOS/Android or as a Desktop App.
- **Works Offline**: Once loaded, use all calculators without an internet connection.
- **Fast & Lightweight**: Premium performance with zero data leaving your device (Privacy First).

### Installation Instructions
- **On Mobile (Chrome/Android)**: Tap the "Install App" button on the home screen or select "Add to Home Screen" from the browser menu.
- **On iOS (Safari)**: Tap the "Share" button and select "Add to Home Screen".
- **On Desktop**: Look for the "Install" icon in the browser address bar.

## 🛠️ Technical Stack

- **Structure**: Semantic HTML5
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Premium Dark Mode UI)
- **Charts**: [Chart.js](https://www.chartjs.org/) for interactive visualizations
- **PWA**: Custom Service Worker and Web Manifest for offline support
- **Fonts**: Inter via Google Fonts

## 💻 Local Development

To run the project locally:

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. To test PWA features locally, use a local server (e.g., `python -m http.server 8000`).
